### 详细步骤

#### 步骤 1：扫描并导出数据（👤 人类操作）

1. **以管理员身份打开 WizTree**
   ```
   F:\coding\xianliao\WizTree\WizTree64.exe
   右键 → 以管理员身份运行
   ```

2. **扫描 C 盘**
   - 点击地址栏，输入 `C:`，按回车
   - 等待扫描完成（通常几秒钟）

3. **导出 CSV**
   - 点击菜单 "文件" → "导出"
   - 勾选选项：
     - ✅ 导出文件夹
     - ✅ 导出驱动器大小信息
     - ❌ 导出文件（不勾选，减小文件大小）
   - 保存到：`F:\coding\xianliao\skills\clean-c-drive\data\`

#### 步骤 2：AI 分析（🤖 自动）

在 Claude Code 中说：

```
帮我分析 C 盘扫描结果
```

或：

```
请阅读 F:\coding\xianliao\skills\clean-c-drive\skill.md 并执行清理流程
```

#### 步骤 3：确认并执行清理（👤 人类操作）

1. 查看 AI 生成的清理报告
2. 以管理员权限运行清理脚本

## 常见问题

### Q1: 为什么管理员权限下 WizTree 可以命令行导出？

**A:** WizTree 使用 NTFS 的 MFT（主文件表）进行快速扫描，读取 MFT 需要管理员权限。命令行导出时也需要这个权限。

### Q2: 清理后会影响系统正常使用吗？

**A:** 不会。高优先级清理的都是缓存和临时文件：
- Windows 更新缓存：更新完成后不再需要
- 浏览器 IndexedDB：视频网站的播放缓存
- 临时文件：系统会自动重建

### Q3: 清理浏览器缓存会丢失数据吗？

**A:** 不会丢失重要数据：
- ✅ 保留：书签、密码、历史记录、登录状态
- ❌ 清理：IndexedDB（视频缓存）、GPUCache（着色器缓存）

### Q4: 多久清理一次比较合适？

**A:** 建议：
- 日常使用：每 2-4 周
- C 盘空间紧张：随时清理
- 安装大型软件前：先清理腾出空间

### Q5: 可以自定义清理规则吗？

**A:** 可以。编辑 `analyze.py` 中的 `CLEANABLE_PATTERNS`：

```python
CLEANABLE_PATTERNS = {
    "high": {
        "patterns": [
            {"pattern": "你的自定义模式", "name": "描述", "safe": True},
            ...
        ]
    }
}
```

## WizTree 命令行参数参考

```powershell
WizTree64.exe "C:" /export="output.csv" /admin=1 /exportfolders=1 /exportfiles=0
```

| 参数 | 说明 |
|------|------|
| `/admin=1` | 使用管理员模式（MFT 扫描） |
| `/export="file.csv"` | 导出到 CSV 文件 |
| `/exportfolders=1` | 导出文件夹 |
| `/exportfiles=0` | 不导出文件（减小文件大小） |
| `/exportdrivecapacity=1` | 导出磁盘容量信息 |
| `/exportmaxdepth=8` | 最大目录深度 |
